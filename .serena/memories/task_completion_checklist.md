# Task Completion Checklist

タスク完了時に実行すべきコマンドと確認事項。

## 1. コード品質チェック

```bash
pnpm lint
```

ESLintエラー・警告がないことを確認。

## 2. ビルド確認

```bash
pnpm build
```

プロダクションビルドが成功することを確認。

## 3. 開発サーバーでの動作確認

```bash
pnpm dev
```

変更した機能が正しく動作することをブラウザで確認。

## 4. セキュリティ確認

- 環境変数がハードコードされていないこと
- `CRYPTO_KEY`、`CRYPTO_IV`などの機密情報が露出していないこと
- Notion資格情報が適切に暗号化されていること

## 5. コミット前の確認

- 不要なconsole.logがないこと
- 不要なコメントアウトされたコードがないこと
- フォーマットが統一されていること

## 注意事項

- このプロジェクトにはテストスイートがない
- Revalidation: 30秒（ISR）なので、変更反映に時間がかかる場合がある
- Production port: 3080（開発は3000）
